# diplom

## Установка

1)[Установить NodeJS](https://nodejs.org/en/);

2)[Установить MongoDB](https://www.mongodb.com/download-center/community);

3)Клонировать репозитории;

4)Запустить mongod;

5)Далее выполнить следующий код в репозитории через консоль.

```sh
$ npm i
$ node server.js
```
